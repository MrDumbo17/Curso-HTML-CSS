Principais formatações

Negrito/Destaque

Nesta frase temos um termo em negrito usando a tag <b> (não semântica).

Nesta frase temos um termo em destaque usando a tag <strong> (semântico).

Itálico/Ênfase

Nesta frase, temos um termo em itálico usando a tag <i> (não semântico).

Nesta frase,temos um termo em ênfase usando a tag <em> (semântico).

Texto marcado

Podemos criar tambem um texto marcado,com a tag <mark> (semântica) e o estilo na própia tag.

E no outro parágrafo temos outro texto marcado, com a mesma tag porém com o estilo geral, configurado no <head>.

Texto grande e pequeno

Extamos criando um texto grande,com tag <big> (não semântica), e um pequeno,com a tag <small> (semântica) nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluido, com a tag <dell> (semântica), para dizer deve ser desconsiderado.

Texto inserido

Podemos marcar um texto como inserido, com a tag <ins> (semântica), para dar uma enfase para consedrar que ele foi adicionado depois.

Ou podemos usar a tag <u> (não semântica) para faze um texto sublinhado.

Texto sobrescrito

Para inserir coisas do tipo x20+3, usamos a tag <sup> (semântica).

Texto sobrescrito

Para inserir coisas do tipo H2O , usamos a tag <sub> (semântica).
