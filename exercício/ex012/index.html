Imágens dinâmicas

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

A imagem acima foi adicionada usando a tag <picture> dentro dela usa-se a tag <img> com os paramentos já explicados anteriormente para selecionar a imágem padrão.

sour

Para torna-la dinamica foi utilizada a tag <source> com parâmetros de media tipe "source:media:tipe".Nos parametros foram usadas as seguintes definições: 

    media="(max-width:'tamano em px'")  - deste modo definiu-se que a imagem sera alternada com base no tamanho maximo da largura em pixels.
    srcset="'diretório da imagem'" = selecionado a imagem q irá, substituí-la.
    type="image/'extenção'" = setado que tipo de imagem estamos trabalhando.
Vale lembrar que o source deve ser definido do menor para o maior e depois a imagem padrão.
